VMware CloudGetting Started HowTo:Manage Users

Using IAM

Users within an Org are managed through Identity & Access Management (IAM).

  1. Click on the Services icon from within the cloud console.
  2. Click on Identity & Access Management.

You will be presented with a list of active users.

Add Users

The following steps will add users to an Organization. Changes are performed from IAM.

  1. Click “Add Users”.
  2. Provide the email address of 1 or more users and specify their roles.
  3. You may optionally set the service roles for the users.
  4. Clicking on “Add” will add the users and cause individual invitation emails to be sent to them.

You may view pending invitations for users which have been added but have not yet activated their account.

Edit Roles

The following steps are for editing user roles within an Organization. Changes are performed from IAM.

  1. Select 1 or more users.
  2. Click “Edit Roles”.
  3. Edit the user’s role within the Organization, add or remove services for the user, or edit the service roles for existing services.
  4. Click “Save” when finished.

Remove Users

The following steps are for removing users from an Organization. Changes are performed from IAM.

  1. Select 1 or more users.
  2. Click “Remove Users”.
  3. Confirm the change.
