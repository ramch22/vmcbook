Add a Network Segment

A network segment is the term used to describe an IP broadcast domain within an SDDC. It is implemented using an NSX Logical Switch.

Network segments are viewed from within the Network & Security tab of the SDDC.

Clicking “Add Segment” will display the creation form for a new network segment.

The following fields are expected: * Name - A name which is meaningful to you. This may be edited later. * Type * Routed - A network segment with a default gateway attached. The most commonly used type of segment. * Extended - Used by the L2VPN feature. * Disconnected - Used by HCX L2 extended networks. * Gateway IP / Prefix Length - For routed networks, specifies the IP/mask to use for the gateway router. May not be edited later. * DHCP - Whether or not to enable native DHCP services. May not be edited later. * DHCP IP Range - Range to use for DHCP assignments. Ignored if DHCP relay is configured. May not be edited later. * DNS Suffix - Domain to be provided with DHCP. May not be edited later.

Save the form to create the network segment.

Edit a Network Segment

Network segments are viewed from within the Network & Security tab of the SDDC.

They are edited from within their context menu.

Delete a Network Segment

Network segments are viewed from within the Network & Security tab of the SDDC.

They are deleted from within their context menu.
