VMware Cloud on AWS

Compute: Instance Types

i3.metal

Specifications

  • CPU - Intel Xeon Processor E5-2686 v4 CPU @ 2.3GHz 2x18 core / 36 thread CPUs, total 72 logical cores
  • Memory - 512GB
  • Storage - 10TB raw storage capacity (8 * 1.7TB NVMe, 2 disk groups, 1 cache disk per group, 5.1TB per group)
  • Network - 25Gbps ENA

AWS reference documentation is here.

Design Considerations
Due to a published vulnerability with this Intel CPU, hyperthreading has been disabled. See this KB for more information.

r5.metal

Specifications

  • CPU - Intel Xeon Skylake 24 Core CPU @2.5GHz
  • Memory - 768GB
  • Storage - EBS-only available in 15, 20, 25, 30, and 35TiB
  • Network - 25Gbps ENA

AWS reference documentation is here.

Design Considerations
  • EBS storage size must be chosen at the time of cluster creation and may be subsequently scaled up, but not scaled down.
  • Discounted pricing for EBS is not available from AWS. This is an important consideration when evaluating this platform.
